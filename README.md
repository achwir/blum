## Installation

Install with VPS/termux

VPS

```bash
  apt update && apt upgrade
```

```bash
  apt install git
```

```bash
  apt install nodejs
```

```bash
  apt install python3
```

```bash
  git clone https://github.com/achwir/blum
```

```bash
  python3 -m venv tomarket
```

```bash
  source tomarket/bin/activate
```

```bash
  pip install -r requirements.txt
```
